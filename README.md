🎤 HablaBien-IA — Entrenador Personal de Oratoria con Inteligencia Artificial
HablaBien-IA es una plataforma web que transforma a estudiantes en oradores convincentes mediante análisis de video impulsado por IA. Combina Procesamiento de Lenguaje Natural, Visión Artificial y Análisis de Sentimiento para detectar muletillas, evaluar lenguaje corporal y medir el tono de voz — todo en tiempo real.
Ideal para egresados universitarios que enfrentan exámenes de grado, entrevistas laborales o presentaciones académicas. 🚀
