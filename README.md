# 📋 Módulo de Historial

## Descripción
Componente de historial de sesiones de práctica de oratoria para HablaBienAI. Permite visualizar el progreso del usuario a través de sus sesiones anteriores.

## Tecnologías
- ⚛️ React
- 🎨 Tailwind CSS
- ⚡ Vite

## Funcionalidades
- Lista de sesiones con fecha, hora, duración y puntaje
- Búsqueda de sesiones en tiempo real
- Niveles de desempeño: Sobresaliente, Bueno, En Progreso
- Indicador de progreso mensual (+15% Mejora)
